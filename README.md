# Windows-API-Template
This is a simple game hacking template using the Windows API

# Methods Information and Usage Guide

This document provides detailed information about the various methods available in our API. Each method is designed to facilitate interaction with game processes and memory management efficiently and effectively. The following table includes descriptions, examples, and method signatures for your reference.

## Method Details

| Method                  | Description                                                                                                    | Example                                                                           | Signature                                                                                  |
|-------------------------|----------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| **FindGameProcess**     | Locates the game process by its executable name. This method returns a handle to the process if found.         | `HANDLE handle = FindGameProcess("GameName");`                                    | `HANDLE FindGameProcess(const std::string &gameName);`                                     |
| **FindGameWindow**      | Identifies the game window by its title. This method returns a handle to the window if found.                  | `HWND gameWindow = FindGameWindow("GameWindowName");`                             | `HWND FindGameWindow(const std::string &windowName);`                                      |
| **ReadAddress**         | Reads a value of a specified type from a given memory address.                                                 | `int value = ReadAddress<int>(0x12345678);`                                       | `template<typename T> T ReadAddress(DWORD address);`                                       |
| **WriteAddress**        | Writes a specified value to a given memory address.                                                            | `bool result = WriteAddress<int>(0x12345678, 100);`                               | `template<typename T> bool WriteAddress(DWORD address, const T &value);`                   |
| **ReadAddressOffset**   | Reads a value of a specified type from a given memory address, including an offset.                            | `int value = ReadAddressOffset<int>(0x12345678, 0x10);`                           | `template<typename T> T ReadAddressOffset(DWORD address, const DWORD offset);`             |
| **WriteAddressOffset**  | Writes a specified value to a given memory address, including an offset.                                       | `bool result = WriteAddressOffset<int>(0x12345678, 0x10, 100);`                   | `template<typename T> bool WriteAddressOffset(DWORD address, DWORD offset, const T &value);`|
| **ReadAddressOffsets**  | Reads a value of a specified type from a given memory address, using a series of offsets.                      | `int value = ReadAddressOffsets<int>(0x12345678, {0x10, 0x20});`                  | `template<typename T> T ReadAddressOffsets(DWORD address, const std::vector<DWORD> &offsets);`|
| **WriteAddressOffsets** | Writes a specified value to a given memory address, using a series of offsets.                                 | `bool result = WriteAddressOffsets<int>(0x12345678, {0x10, 0x20}, 100);`          | `template<typename T> bool WriteAddressOffsets(DWORD address, const std::vector<DWORD> &offsets, const T &value);`|
| **ReadPointer**         | Reads a value of a specified type from a pointer address.                                                      | `int value = ReadPointer<int>(0x12345678);`                                       | `template<typename T> T ReadPointer(DWORD address);`                                       |
| **WritePointer**        | Writes a specified value to a pointer address with an additional offset.                                       | `bool result = WritePointer<int>(0x12345678, 0x10, 100);`                         | `template<typename T> bool WritePointer(DWORD address, DWORD pointerOffset, const T &value);`|
| **ReadPointerOffset**   | Reads a value of a specified type from a pointer address with an offset.                                       | `int value = ReadPointerOffset<int>(0x12345678, 0x10);`                           | `template<typename T> T ReadPointerOffset(DWORD address, const DWORD offset);`             |
| **WritePointerOffset**  | Writes a specified value to a pointer address with an offset and additional offset.                            | `bool result = WritePointerOffset<int>(0x12345678, 0x10, 0x20, 100);`             | `template<typename T> bool WritePointerOffset(DWORD address, DWORD pointerOffset, DWORD offset, const T &value);`|
| **ReadPointerOffsets**  | Reads a value of a specified type from a pointer address using multiple offsets.                               | `int value = ReadPointerOffsets<int>(0x12345678, {0x10, 0x20});`                  | `template<typename T> T ReadPointerOffsets(DWORD address, const std::vector<DWORD> &offsets);`|
| **WritePointerOffsets** | Writes a specified value to a pointer address using multiple offsets.                                          | `bool result = WritePointerOffsets<int>(0x12345678, 0x10, {0x20, 0x30}, 100);`    | `template<typename T> bool WritePointerOffsets(DWORD address, DWORD pointerOffset, const std::vector<DWORD> &offsets, const T &value);`|
| **ReadString**          | Reads a string from a specified memory address. The size parameter defines the length of the string.           | `std::string value = ReadString(0x12345678, 20);`                                 | `std::string ReadString(DWORD address, size_t size);`                                      |
| **WriteString**         | Writes a string to a specified memory address.                                                                 | `bool result = WriteString(0x12345678, "Hello World");`                           | `bool WriteString(DWORD address, const std::string &str);`                                 |
| **HotKeysDown**         | Checks if a combination of specified keys are currently pressed.                                               | `bool isPressed = HotKeysDown({VK_A, VK_B, VK_C});`                               | `bool HotKeysDown(const std::vector<int> &keys);`                                          |
| **IsKeyPressed**        | Checks if a specific key is currently pressed.                                                                 | `bool isPressed = IsKeyPressed(VK_A);`                                            | `bool IsKeyPressed(int keycode);`                                                          |
| **IsKeyToggled**        | Checks if a specific key is currently toggled.                                                                 | `bool isToggled = IsKeyToggled(VK_CAPITAL);`                                      | `bool IsKeyToggled(int keycode);`                                                          |
| **GetGameName**         | Returns the name of the currently active game.                                                                 | `std::string gameName = GetGameName();`                                           | `std::string GetGameName();`                                                               |
| **GetProcessId**        | Returns the process ID of the currently active game.                                                           | `DWORD processId = GetProcessId();`                                               | `DWORD GetProcessId();`                                                                    |
| **GetGameHandle4mHWND** | Retrieves the game handle using the window handle.                                                             | `HANDLE gameHandle = GetGameHandle4mHWND(gameWindow);`                            | `HANDLE GetGameHandle4mHWND(HWND hwnd);`                                                   |
| **GetProcessID4mHWND**  | Retrieves the process ID using the window handle.                                                              | `DWORD processId = GetProcessID4mHWND(gameWindow);`                               | `DWORD GetProcessID4mHWND(HWND hwnd);`                                                     |
| **GetGameHandle**       | Returns the handle of the currently active game.                                                               | `HANDLE gameHandle = GetGameHandle();`                                            | `HANDLE GetGameHandle();`                                                                  |
| **GetGameBaseAddress**  | Returns the base memory address of the currently active game.                                                  | `DWORD baseAddress = GetGameBaseAddress();`                                       | `DWORD GetGameBaseAddress();`                                                              |
| **EnableLogs**          | Enables or disables logging functionality. Useful for debugging and tracking operations.                       | `EnableLogs(true);`                                                               | `void EnableLogs(bool status);`                                                            |
| **ReadCheatTable**      | Reads a cheat table file and returns a structured representation of the cheats. The optional entries parameter limits the number of entries read. | `CheatTable ct = ReadCheatTable("cheatTableFile.ct");`                            | `CheatTable ReadCheatTable(const std::string &cheatTableFile, int entries = -1);`          |
| **AddCheatTableEntry**  | Adds an entry to the cheat table, specifying description, data type, address, offsets, hotkeys, action, and value. | `AddCheatTableEntry("Armor", CheatTypes.Integer, 0x07290BC8, {}, {VK_CONTROL, 'M'}, CheatActions.SetValue, "150");`| `void AddCheatTableEntry(const std::string &description, const std::string &dataType, DWORD address, const std::vector<DWORD> &offsets, const std::vector<int> &hotkeys, const std::string &hotkeyAction, const std::string hotkeyValue);`|
| **DisplayCheatTable**   | Displays the current cheat table with optional parameters to show or hide various columns.                     | `DisplayCheatTable();`                                                            | `void DisplayCheatTable(bool showMenuIndex = true, bool showMenuDescription = true, bool showMenuAction = false, bool showMenuHotkeys = true, bool showMenuValue = false);`|
| **ReadCheatTableEntries** | Reads and outputs the values of all cheat entries in the cheat table.                                        | `ReadCheatTableEntries();`                                                        | `void ReadCheatTableEntries();`                                                            |
| **ActivateCheatTableEntries** | Activates specified cheat table entries based on their indices.                                          | `ActivateCheatTableEntries({1, 2, 3});`                                           | `void ActivateCheatTableEntries(const std::vector<int> &cheatEntryIndex);`                 |
| **ExecuteCheatTable**   | Executes the cheat table and optionally generates a trainer with specified parameters.                         | `ExecuteCheatTable(true, VK_ESCAPE, true, true, false, true);`                    | `void ExecuteCheatTable(bool showTrainerOutput = false, int exitTrainerKey = EXIT_TRAINER_KEY, bool showMenuIndex = true, bool showMenuDescription = true, bool showMenuAction = false, bool showMenuHotkeys = true);`|
